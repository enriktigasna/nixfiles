## Nixfiles
Features:
- Flake setup with Home manager
- GNOME
- CTF packages (pwntools, pwninit, ghidra, sage etc.)
- Neovim (nixvim) config with LSP, formatters, QoL
- Generally nice QoL setup with prompt, nerdfont, zellij etc.
